# NumPy-Yahtzee
Command Line Yahtzee. Made with NumPy. 


SAMPLE:
```
1st Roll:


-----   -----   -----   -----   -----   
|o  |   |o  |   |o  |   |o o|   |o  |   
|   |   | o |   |   |   |o o|   | o |   
|  o|   |  o|   |  o|   |o o|   |  o|   
-----   -----   -----   -----   -----   
    2       3       2       6       3


Possible Plays
---------------
Doubles of: 2
 
Doubles of: 3
 
Enter dice to roll (1-5) - Press 'n' to roll:4
Enter dice to roll (1-5) - Press 'n' to roll:n
 
2nd Roll:


-----   -----   -----   -----   -----   
|o  |   |o  |   |o  |   |o o|   |o  |   
|   |   | o |   |   |   |o o|   | o |   
|  o|   |  o|   |  o|   |o o|   |  o|   
-----   -----   -----   -----   -----   
    2       3       2       4       3


 
Possible Plays
---------------
Doubles of: 2
 
Doubles of: 3
 
Enter dice to roll (1-5) - Press 'n' to roll:4
Enter dice to roll (1-5) - Press 'n' to roll:n
 
3rd Roll:


-----   -----   -----   -----   -----   
|o  |   |o  |   |o  |   |o  |   |o  |   
|   |   | o |   |   |   |   |   | o |   
|  o|   |  o|   |  o|   |  o|   |  o|   
-----   -----   -----   -----   -----   
    2       3       2       2       3


 
Possible Plays
---------------
3 of a Kind -  2
 
Doubles of: 3
 
Full House
```
